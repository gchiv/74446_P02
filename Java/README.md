# Metricas de Rendimiento:

## Datos

- Tiempo Secuencial (T_s): 39 ms
- Tiempo Paralelo (T_p): 23 ms
Procesadores (p): 4

## MÃ©tricas de Rendimiento

1. Speedup (S)
S = T_s / T_p
S = 39 ms / 23 ms = 1.70x

2. Eficiencia (E)
E = S / p
E = 1.70 / 4 = 0.425 (42.5%)

3. Overhead (T_o)
T_o = p * T_p - T_s
T_o = (4 * 23 ms) - 39 ms = 53 ms
